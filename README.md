## 🚀 使用方式

```
遇到问题时 → 找到对应文档 → 丢给 AI → 按步骤执行
```

**每个文档的结构：**
- 🎯 **问题现象** — 一眼确认是不是同一个问题
- 🔍 **根因分析** — 为什么会这样
- ✅ **解决方案** — 按步骤操作即可
- 📝 **应急备忘** — 关键命令 / 关键路径，不用翻正文
